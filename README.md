## Navbar
До входу:
- лого (силка на home)
- Назва (силка на home)
- Пошук по темах/запитаннях
- 2 кнопки - зайти і реєстрація
- Кнопка спливаючого вікн: налаштування мови і змінат теми

Після входу:
- лого (силка на home)
- Назва (силка на home)
- Пошук по темах/запитаннях
- Чат друзів
- збережені теми
- Кнопка спливаючого вікна:
  - налаштування мови
  - зміна теми
  - налуштування профіля
  - Налштування
  - Help
  - Logout
